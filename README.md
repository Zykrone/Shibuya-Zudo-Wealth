## 🌌 LE CONCEPT : SHIBUYA INVEST
**Shibuya Zudo Wealth** n'est pas qu'une simple boutique, c'est une **Extension du Domaine** dédiée à l'investissement et à la gestion de privilèges. Inspiré par l'esthétique sombre et néon de *Jujutsu Kaisen*, la plateforme offre une expérience utilisateur luxueuse et immersive pour acquérir des rôles de prestige et des accès exclusifs.

---

## 🔥 FONCTIONNALITÉS "DOMAIN EXPANSION"

### 🔒 Panel Admin Secret (Werenoi System)
Accès réservé aux détenteurs du code secret via un point d'entrée discret.
- **Interface Glassmorphism** : Un panel futuriste centré avec effets de flou et néons.
- **Real-time Discount Control** : Activez/Désactivez les réductions globales ou spécifiques d'un simple clic.
- **User Management** : Contrôle total sur les rôles et les permissions des membres.

### 🏷️ Système de Réductions Dynamique
- **Global Roles (-20%)** : Promotions automatiques sur l'ensemble du catalogue.
- **Tiered Discounts** : Réductions spécifiques pour les grades *Couronne* (-15%) et *Créateur* (-10%).
- **Whitelist Protection** : Remises automatiques sur les packs *Owner* et *Sys*.

### 💎 Expérience Premium
- **Investment Grade Table** : Une visualisation haut de gamme des grades de prestige avec effets de survol et auras lumineuses.
- **Audio Immersif** : Système audio automatique pour une immersion totale dès l'entrée dans le domaine.
- **Dynamic Pricing** : Affichage intelligent des prix barrés et des badges de réduction.

---

## 🛠️ STACK TECHNIQUE
- **Core** : React.js (Vite)
- **State Management** : Context API (Auth & Discounts)
- **Styling** : Vanilla CSS 3 (Custom Design System, Glassmorphism, Neon Glow)
- **Database** : LocalStorage Persistence (Simulation de DB temps réel)

---

## 🚀 INSTALLATION

1. Clonez le domaine :
   ```bash
   git clone https://github.com/Zykrone/Shibuya-Zudo-Wealth.git
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Déployez le domaine :
   ```bash
   npm run dev
   ```
